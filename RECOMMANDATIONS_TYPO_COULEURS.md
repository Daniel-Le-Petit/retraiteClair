## 💡 Conseils Techniques PowerPoint

### **1. Créer un Fond Semi-Transparent**
```
1. Insérer un rectangle
2. Format → Remplissage → Solide
3. Couleur : Bleu foncé (#0F172A)
4. Transparence : 25% (donne 75% d'opacité)
5. Ligne : Aucune ligne
```

### **2. Ajouter une Ombre Portée**
```
1. Sélectionner le texte
2. Format → Effets de texte → Ombre
3. Options d'ombre :
   - Couleur : Noir (#000000)
   - Transparence : 60%
   - Flou : 6pt
   - Distance : 3pt
   - Angle : 90° (en bas)
```

### **3. Hiérarchie Visuelle**
- **Titre :** Taille la plus grande, gras, ombre forte
- **Points clés :** Taille moyenne, semi-gras, ombre légère
- **Sous-titre :** Taille normale, gras pour "RETRAITE PROGRESSIVE"
- **CTA :** Taille petite mais dans un bouton coloré
